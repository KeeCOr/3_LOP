# LOP (Land of Power) 설계 문서

> 작성일: 2026-04-21

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 전략 보드게임 (모노폴리 기반) |
| 플랫폼 | 브라우저 (Next.js), 추후 Electron + 온라인 멀티 |
| 핵심 루프 | 주사위 굴림 → 말 선택 → 이동 → 칸 이벤트 → 전투/건설/구매 |
| 승리 조건 | 상대방을 파산(돈 0)시키기 |
| 패배 조건 | 돈이 0이 되면 탈락 |
| 플레이어 수 | 1인 vs AI (추후 온라인 멀티 확장) |

## 2. 기술 스택

- **Next.js + React + TypeScript**
- **Tailwind CSS** — UI 스타일링
- 상태관리: React `useReducer` (외부 라이브러리 없음)
- **Electron** — 추후 데스크탑 빌드

## 3. 게임 플로우

```
게임 시작 → 캐릭터 선택 → 보드 시작
  └→ [매 턴]
       ├→ 주사위 굴림
       ├→ 이동할 말 1개 선택
       ├→ 선택한 말 이동
       └→ 칸 이벤트 처리
            ├→ 중립 땅: 전투 또는 골드 지불로 구매
            ├→ 내 땅: 건물 건설/업그레이드 여부 선택
            ├→ 적 땅: 전투 또는 통행세 납부
            ├→ 적 말 조우: 말 전투 시뮬레이션
            ├→ 상점: 병력·말·장비 구매
            ├→ 세금: 보유 돈의 10% 납부
            ├→ 찬스/커뮤니티: 랜덤 이벤트 카드
            └→ 출발점 통과: 병력 자동 보충
```

## 4. 보드 레이아웃

5×4 외곽 14칸 구성. 플레이어(P)와 AI(E)는 대각선 반대편에서 시작.

```
[ P ] [땅1] [땅2] [상점] [땅3]
[땅4]                   [찬스]
[세금]                  [땅5]
[땅6] [커뮤] [땅7] [땅8] [ E ]
```

| 칸 종류 | 수 | 설명 |
|---------|-----|------|
| 출발 (P/E) | 2 | 각자 시작점. 통과 시 병력 보충 지급 |
| 일반 땅 | 8 | 게임 시작 시 중립 병력 배치. 전투 또는 구매로 점령 |
| 상점 | 1 | 병력·말·장비 구매 |
| 세금 | 1 | 보유 돈의 10% 납부 |
| 찬스 | 1 | 랜덤 이벤트 카드 (개인 행운/불운) |
| 커뮤니티 | 1 | 랜덤 이벤트 카드 (전체 영향) |

**찬스 카드 예시** (6종 이상):
- 골드 200 획득
- 병력 5명 즉시 지급
- 다음 전투 공격력 1.5× (1회)
- 가장 가까운 상점으로 이동
- 세금 면제 1회
- 적 땅 1곳 통행세 지불 면제 1회

**커뮤니티 카드 예시** (6종 이상):
- 모든 플레이어 골드 100 지급
- 모든 플레이어 병력 3명 지급
- 다음 한 바퀴 통행세 2배
- 다음 한 바퀴 건물 건설 비용 50% 할인
- 랜덤 땅 1곳의 소유권 초기화 (무주지로 변환)
- 주사위 결과 +2 보너스 (다음 턴)

## 5. 말(장수) 시스템

### 5.1 기본 능력치 (고정)

| 능력치 | 역할 |
|--------|------|
| `공격력` | 전투 시 병력 데미지 배율 |
| `방어력` | 받는 데미지 감소 |
| `통솔력` | 데리고 다닐 수 있는 최대 병력 수 결정 |
| `이동력` | 주사위 결과에 +보너스 |

게임 시작 시 캐릭터 4종 중 1개 선택. 능력치는 고정이며 장비로만 강화 가능.

| 캐릭터 | 공격력 | 방어력 | 통솔력 (최대 병력) | 이동력 |
|--------|--------|--------|------------------|--------|
| 장군 | 1.3× | 0.8× | 20명 | +0 |
| 기사 | 1.1× | 1.2× | 15명 | +1 |
| 상인 | 0.9× | 0.9× | 12명 | +0 |
| 척후 | 1.0× | 1.0× | 10명 | +2 |

### 5.2 장비 슬롯

- 말마다 슬롯 3개 (장착 장비 없으면 빈 슬롯)
- 상점·찬스카드에서 획득

| 장비 | 효과 | 상점 가격 |
|------|------|----------|
| 검 | 공격력 +0.2× | 400 |
| 갑옷 | 방어력 +0.2× | 400 |
| 군기 | 최대 병력 +8명 | 350 |
| 군화 | 이동력 +1 | 300 |

### 5.3 말 고용

- 상점 칸에서만 추가 고용 가능
- 고용 즉시 해당 플레이어의 출발점에서 등장

| 고용 순서 | 비용 |
|----------|------|
| 1번째 | 무료 (시작 지급) |
| 2번째 | 500 |
| 3번째 | 1,200 |
| 4번째 | 2,500 |
| 5번째 | 5,000 |
| N번째 | 이전 비용 × 2 + 300 |

### 5.4 턴 진행

- 매 턴 주사위를 한 번 굴림
- 보유한 말 중 **1개를 선택**해서 이동
- 전투 패배 시 해당 말은 병력 0으로 출발점 귀환 (소멸 없음)

## 6. 병력 시스템

| 항목 | 내용 |
|------|------|
| 구매 | 상점 칸 방문 시 1명당 50골드 |
| 최대 보유 | 말의 통솔력에 따라 결정 |
| 출발점 보상 | 출발점 통과 시 +5명 자동 지급 (말이 여러 개면 각각 적용) |
| 영토 보충 | 한 바퀴 완주 시 병영 건물이 있는 영토에서 자동 생산 |

- 말이 이동할 때 데리고 다니는 병력만 전투에 참여
- 땅에 배치된 병력은 수비 전용

---

## 6-1. 중립 병력 & 땅 점령 시스템

### 게임 시작 시 중립 병력 배치

8개 일반 땅 모두 게임 시작 시 중립 병력이 무작위 배치됨:

| 땅 위치 | 중립 병력 범위 |
|---------|-------------|
| P 출발점 인근 (1~3칸) | 3 ~ 6명 |
| 중간 구간 (4~6칸) | 5 ~ 10명 |
| E 출발점 인근 (7~8칸) | 3 ~ 6명 |

중립 병력은 공격 없이 수비만 함 (AI처럼 행동하지 않음).

### 땅 점령 방법 (2가지 선택)

중립 땅 또는 적 땅 착지 시 선택:

| 방법 | 조건 | 결과 |
|------|------|------|
| **전투** | 병력 보유 | 시뮬레이션 진행. 승리 시 점령 |
| **골드 구매** | 충분한 돈 | 전투 없이 즉시 점령. 비용 = 중립 병력 수 × 80골드 |

> 적 땅은 전투만 가능. 골드 구매는 중립 땅에서만 허용.

### 점령 시 병력 배치

전투 승리 또는 골드 구매로 점령 직후:

1. **기본 수비 병력 자동 배치**: 점령 방식에 따라 자동으로 일부 병력 배치
   - 전투 승리: 남은 아군 병력 중 말이 직접 배치 수량 선택 (최소 1명)
   - 골드 구매: 구매한 중립 병력이 그대로 내 수비대로 전환

2. **추가 배치**: 말이 보유한 병력 중 원하는 만큼 추가 배치 가능 (말 보유 병력 감소)

## 7. 건물 시스템

땅 구매 후, **자기 땅 칸에 착지했을 때** 건설/업그레이드 선택 가능.  
땅 1칸당 1종류 선택. 같은 종류로만 Lv3까지 업그레이드.  
땅을 빼앗기면 건물째 새 주인에게 귀속.

### 🏦 경제 건물 — 금고 (돈 생산)

| 등급 | 비용 | 한 바퀴 수입 | 통행세 |
|------|------|------------|--------|
| Lv1 금고 | 200 | +100 | 100 |
| Lv2 은행 | 500 | +250 | 200 |
| Lv3 왕실 금고 | 1,000 | +500 | 350 |

### ⚔️ 병력 건물 — 병영 (병력 생산)

| 등급 | 비용 | 한 바퀴 생산 | 통행세 |
|------|------|------------|--------|
| Lv1 병영 | 200 | +3명 | 100 |
| Lv2 훈련소 | 500 | +7명 | 200 |

### 🏰 강화 건물 — 초소 (수비 강화)

| 등급 | 비용 | 수비 보너스 | 통행세 |

| Lv2 성벽 | 500 | 공격+35%, 방어+25% | 450 |

## 8. 전투 시스템

### 8.1 전투 발생 조건

| 상황 | 설명 |
|------|------|
| **영토 전투** | 적 땅에 착지 → 그 땅의 수비 병력과 전투 |
| **말 전투** | 같은 칸에 적 말과 겹침 → 말끼리 + 보유 병력으로 전투 |

### 8.2 전투 vs 통행세 선택

적 땅 착지 시 선택:
- **전투**: 시뮬레이션 진행, 승리 시 점령
- **통행세**: 건물 등급에 따른 고정 금액 납부 (전투 없이 통과)

### 8.3 전투 계산

- 매 라운드: 공격측 병력 × 말의 공격력 배율 → 적 병력 차감 (방어력으로 감소)
- 수비측도 동시 반격
- 강화 건물이 있으면 수비 병력의 공격력/방어력 추가 보너스 적용
- 병력이 먼저 0이 되는 쪽 패배

**승리 시**: 공격자가 남은 병력 중 1명 이상 배치해야 점령 확정  
**패배 시**: 말이 병력 0으로 출발점 귀환

### 8.4 전투 시뮬레이션 화면

모달 창이 열리며 실시간으로 전투 진행 표시:

```
[공격측]              VS         [수비측]
 말 이름                          말 이름 / 수비대
 병력: 15명                       병력: 8명
 공격력 1.3×                      방어력 0.9×

 ████████░░  HP                  ████░░░░░░  HP

 [전투 로그]
 > 아군 3명이 적 2명을 쓰러뜨렸다
 > 적 반격: 아군 1명 전사
 > ...
```

## 10. AI 시스템

### 난이도 3단계

| 난이도 | 행동 방식 |
|--------|----------|
| 쉬움 | 이동할 말 랜덤 선택. 구매·건설은 랜덤 확률 결정 |
| 보통 | 상황별 우선순위 규칙 적용 |
| 어려움 | 보통 규칙 + 플레이어 자원 상태 파악 후 최적 행동 |

### 보통·어려움 AI 우선순위 규칙

| 상황 | AI 행동 |
|------|--------|
| 빈 땅 착지 | 돈 여유 있으면 구매 |
| 자기 땅 착지 | 병력 부족 시 병영, 돈 부족 시 금고 우선 건설 |
| 적 땅 착지 | 병력 우세 시 전투, 열세 시 통행세 |
| 돈 부족 | 병력 구매 자제, 통행세 선택 |
| 영토 3개+ (어려움) | 2번째 말 고용 고려 |
| 플레이어 약세 (어려움) | 가장 약한 영토 집중 공략 |

### AI 턴 처리 순서
1. 주사위 굴림
2. 규칙에 따라 이동할 말 1개 선택
3. 이동 및 칸 이벤트 자동 처리
4. 전투 발생 시 시뮬레이션 화면 표시 (플레이어도 관전)
